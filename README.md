# -Fuel-efficiency-prediction

**summary of the project**
The workbook studies car data using the ISLP package. It checks features like mileage, weight, horsepower, and model year. The data was cleaned and analyzed using charts and statistics. It shows that lighter and newer cars give better mileage. The workbook helps to understand and predict fuel efficiency.

The Auto dataset contains information about different cars.
It includes details like mileage (mpg), cylinders, engine size, horsepower, weight, acceleration, model year, origin, and car name.
Most variables are numerical, and a few are categorical.
The dataset is mainly used to study and predict fuel efficiency (mpg).
